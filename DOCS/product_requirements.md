# 产品需求摘要

PitGuard BIM Designer 面向岩土工程师、结构工程师、基坑支护设计人员和 BIM 工程师，提供从地勘数据到围护结构初选、规范子集复核、IFC 和计算书输出的本地优先工作流。

engineering-v0.2 覆盖：

- 项目管理与本地 SQLite JSON 存储。
- 钻孔 CSV/XLSX 导入、校验与地层参数合并。
- IDW 三维地质面、VTU ASCII XML 网格导入。
- 基坑轮廓、边段、代表性剖面。
- 地下连续墙、冠梁、腰梁、支撑和立柱自动初选。
- JGJ120/GB50010/GB50009/GB50007/GB50017 规则库子集筛查。
- 土压力、水压力、支撑轴力、墙体内力、配筋建议。
- Three.js 前端三维可视化骨架：图层开关、透明度、剖切、对象拾取、属性面板。
- 工程级 IFC4 STEP 导出和 DOCX 计算书。
- 完整示例项目，可一键跑通从导入到导出的流程。
