# PitGuard BIM Designer V3.3.0

面向基坑围护结构方案设计、分阶段计算、规范子集审查、配筋设计、三维复核及 IFC/CAD/计算书交付的工程设计辅助系统。

## 启动

- Windows：运行 `start-windows.bat`
- Linux/macOS：运行 `bash start-linux.sh`
- 前端默认地址：`http://127.0.0.1:5173`
- API 文档：`http://127.0.0.1:8000/docs`

系统使用当前 Python 环境，不自动创建额外虚拟环境。前端依赖已锁定，建议使用 `npm ci`。

## 文档入口

全部说明文档集中在 [`docs/README.md`](docs/README.md)。根目录仅保留启动脚本、源码目录和本文件。

## V3.3.0 关键变化

- 增加设计寿命、湿度、持续荷载、徐变、收缩、温差和抗震等级参数，按墙体配筋区输出长期位移和裂缝控制。
- 支撑拓扑按层检查连通性、双向传力、墙面覆盖率、关节点、桥接杆件和凹角，并生成避障加撑候选。
- 碰撞检查扩展到跨层截面包络竖向净距、施工障碍保护区和钢筋笼拥挤。
- 节点局部筛查升级为三自由度凝聚刚度模型，输出偏心转动、特征值稳定性、荷载路径和非线性有限元复核标记。
- 监测数据支持中英文 CSV，单位自动归一；反演系数实际进入墙体与全局联立计算，应用后旧结果自动失效。
- 四级审签增加岗位分离、退回意见、快照失效和失效后重新提交。
- 施工版正式图纸要求工程闸门、当前快照四级批准和施工版修订记录同时满足。
- 工作台保持精简/专业双模式，增加工程深化分组、快捷命令、监测草稿、键盘导航和无障碍状态播报。

## 验证

- 后端快速门禁：`bash scripts/test-backend.sh fast`
- 后端完整隔离门禁：`bash scripts/test-backend.sh full-isolated`
- 前端测试与构建：`bash scripts/test-frontend.sh`

Windows 后端门禁可运行 `powershell -ExecutionPolicy Bypass -File scripts/test-backend.ps1 -Mode fast`。

## 工程边界

系统输出用于方案设计、工程复核和成果组织。生产级施工图签发仍需注册岩土/结构工程师对参数、计算模型、规范适用性、节点构造、施工工序和现场条件进行审查。
