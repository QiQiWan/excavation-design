# 企业工程资源库

`packages/enterprise/*.json` 保存项目可选择的企业/地方标准模板、节点大样目录、钢筋组合和出图规则。

正式项目使用要求：

1. 由企业总工办或项目技术负责人确认 `libraryId`、`libraryVersion` 和适用范围；
2. 安全目标不得低于项目采用规范和专项审查要求；
3. 节点模板必须给出截面适用范围、轴力范围和标准图编号；
4. 钢筋组合必须与混凝土等级、钢筋等级、净距、套筒产品和施工能力匹配；
5. 修改资源库后应重新确认设计基准并重新运行计算、配筋和节点深化；
6. 软件内置库状态为 `software_validation`，不能直接替代企业批准的正式资源库。
