# V2.6.0 案例计算修复、公式显示、导出收敛、钢筋形态和墙体云图

## 1. 一键复核计算卡死/无响应修复

上传案例包含多个历史计算结果和候选方案记录。旧流程在每次运行规范计算前都会重新执行支撑候选枚举，枚举阶段对完整 Project 做深拷贝，导致大项目在候选搜索中长时间停留。V2.6.0 修改为：

- 已存在有效候选修复摘要且当前支撑布置未失败时，规范计算直接复用现有支撑方案；
- 用户主动点击“优化支撑候选”时才重新枚举候选；
- 候选搜索改为浅拷贝 Project，并清空历史 calculationResults / calculationCases，仅深拷贝当前 retainingSystem。

上传案例本地复核：`run_calculation` 约 0.8 s 返回，不再卡在支撑优化深拷贝。

## 2. 计算追溯链公式显示优化

前端新增基于 KaTeX 的 `FormulaDisplay`，将常见公式字符串渲染为可读数学表达，例如 `gamma_0 * gamma_F * envelope(M_stage)` 会显示为带希腊字母、运算符和变量样式的公式块，避免直接显示原始程序字符串。

## 3. 面向使用者的界面收敛

- 计算追溯链标题和说明压缩；
- CAD 模板、导出卡片、钢筋面板去掉版本化开发说明；
- Step 8 导出入口改为 4 个常用成果优先：IFC、CAD、DOCX、钢筋数据；
- 其他调试型成果收纳到“更多导出”。

## 4. 钢筋可视化形态修复

钢筋 IFC 可视化接口新增 `points` 与 `shapeKind` 字段。前端不再只按 start/end 画直线，而按折线段绘制：

- 纵筋增加搭接偏移折线；
- 分布筋增加端部弯钩折线；
- 梁/支撑箍筋按闭合矩形箍筋显示；
- 节点箍筋按局部闭合环显示；
- 钢筋详图导出的 individualBars 同步保留折线形态。

## 5. 围护墙云图

ResultViewer 新增“围护墙变形与内力云图”模块，放在关键内力包络之后。交互方式：

- 指标切换：变形 / 弯矩 / 剪力；
- 平面控制云图：按每面墙控制值着色；
- 深度条带云图：显示前 6 面墙沿深度的控制值分布；
- issue locator 可高亮目标墙段。

该位置紧邻原有内力包络，用户可先看曲线，再看平面/墙身控制区，认知路径更顺。
