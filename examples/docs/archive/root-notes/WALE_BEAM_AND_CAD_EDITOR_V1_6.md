# V1.6.0 围檩连续梁反力分配与 CAD-like 基坑轮廓编辑器

## 1. 本轮目标

V1.6.0 的目标是解决两个关键问题：

1. 支撑轴力分配从 V1.5 的墙面 tributary width 估算升级为“围檩连续梁—弹性支撑节点反力”计算。
2. 基坑轮廓输入从表格输入升级为 CAD-like 交互编辑器，降低建模门槛。

## 2. 围檩连续梁—支撑节点反力算法

### 2.1 计算思路

正常基坑内支撑体系中，墙面侧向压力通常先传给围檩，围檩再通过节点传给水平支撑。V1.6.0 按这一传力路径建模：

```text
墙面土压力/水压力
→ 对应开挖阶段的竖向压力带积分
→ 墙面线荷载 q(kN/m)
→ 围檩连续梁
→ 支撑端部弹性支座
→ 节点反力 R
→ 支撑轴力 N = R / cos(theta)
```

其中 `theta` 通过支撑轴线与墙面法线方向的夹角确定。主对撑的法向投影通常接近 1，角撑和斜撑根据投影折减刚度并放大轴力换算。

### 2.2 梁单元模型

每一段墙面、每一道支撑标高形成一个一维围檩连续梁模型。梁节点包括：

1. 墙面起点。
2. 墙面终点。
3. 所有连接在该墙面的支撑节点。

围檩采用 Euler-Bernoulli 梁单元，每个节点包括横向位移和转角两个自由度。墙面线荷载作为均布荷载转化为等效节点荷载。支撑端部作为横向弹簧，弹簧刚度按：

```text
k = E * A / L * alpha_n^2 * role_factor
```

其中：

- `E` 为支撑材料弹性模量。
- `A` 为支撑截面面积。
- `L` 为支撑跨长。
- `alpha_n` 为支撑轴线对墙面法向的投影系数。
- `role_factor` 用于区分主对撑、角撑、环撑的等效刚度贡献。

### 2.3 反力与轴力

求解梁单元刚度方程后，每个支撑节点的法向反力为：

```text
R_i = k_i * u_i
```

对应支撑轴力为：

```text
N_i = R_i / alpha_n
```

随后再按现有 GB50009/JGJ120 子集的设计效应放大系数生成 `axialForceDesign`。

### 2.4 退化机制

当围檩连续梁模型条件不足或矩阵求解失败时，系统退化为 V1.5 的 tributary width 分配，并在 `distributionMethod` 和 `distributionNote` 中标记。

## 3. 数据输出

`SupportForceResult` 新增字段：

```text
faceCode
supportEndpoint
waleBeamCode
waleChainage
tributaryWidth
continuousBeamReaction
elasticSupportStiffness
normalProjectionFactor
beamNodeCount
distributionMethod
distributionNote
```

前端计算结果页会展示围檩连续梁节点反力表。

## 4. CAD-like 基坑轮廓编辑器

V1.6.0 将 `ExcavationEditor` 从表格录入升级为交互式绘图组件，支持：

1. 拖拽轮廓点。
2. 点击边中点插入新点。
3. 删除选中点。
4. 撤销/重做。
5. 滚轮缩放。
6. Alt/中键拖动画布。
7. 网格显示。
8. 网格吸附。
9. 正交约束。
10. Fit view。
11. 前端实时校验自交、短边和标高错误。

## 5. 当前边界

V1.6.0 的围檩连续梁模型仍属于工程设计辅助模型。后续仍需补充：

1. 围檩自身弯矩、剪力和配筋。
2. 支撑-围檩节点局部三维实体节点分析。
3. 支撑预加轴力、温度作用和施工偏心。
4. 地下室楼板换撑后的刚度转换。
5. 多道支撑之间的空间耦合。
6. 更严格的规范条文映射与正式计算书表达。
