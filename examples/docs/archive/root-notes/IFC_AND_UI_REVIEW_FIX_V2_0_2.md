# V2.0.2 IFC 与交互修复说明

## 审阅结论

上传成果中的计算书内容基本完整，但支撑平面布置偏稀疏，50m 方向仅布置两个主对撑，分仓间距约 16.7m，不符合常规 3-6m 的施工与布置习惯。IFC 文件可见实体数量较多，但 STEP 字符串包含原始中文 UTF-8，严格 IFC 查看器可能解析失败或无法可视化。

## 修复项

1. IFC 输出将非 ASCII 文本编码为 IFC STEP `\X2\...\X0\`，提高 BIM 查看器兼容性。
2. 主对撑沿长向分仓间距由 18m 调整为 5m，并约束在 3-6m 工程常用范围。
3. 未锁定绝对坐标时，基坑轮廓保存后自动与地质模型 XY 范围中心对齐。
4. 前端主流程减少按钮，单独操作移入右侧高级抽屉。
5. CAD 编辑器拖动点时保持网格显示，并增加蓝色水平/竖向对齐参考线。

## 后续建议

下一步应把“支撑间距”做成可配置设计参数，并在前端提供支撑分仓预览、支撑净距检查、出土口避让复核和支撑布置合理性评分。
