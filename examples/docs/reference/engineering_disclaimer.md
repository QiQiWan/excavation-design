# 工程免责声明

本软件当前版本为 engineering-v0.2 原型。软件根据输入资料和当前规则库自动生成的结果用于方案设计、教学演示、参数复核和技术比选辅助，不应替代注册岩土工程师、注册结构工程师或具有相应资质专业人员的判断。

施工图设计、专家论证和正式报审前，应由专业人员逐项复核：

1. 地勘资料、地下水资料和试验参数的可靠性。
2. 规范适用性、设计等级、周边环境等级、荷载组合和地方规定。
3. 支护结构形式、施工阶段、支撑刚度、预加力、拆换撑和节点构造。
4. 地连墙嵌固、整体稳定、坑底隆起、抗渗流/突涌、变形控制和周边沉降。
5. 混凝土构件承载力、裂缝、锚固、搭接、钢筋净距、保护层和施工图详图。
6. IFC/BIM 模型与设计图纸、计算书、工程量清单之间的一致性。

软件计算书中的 pass 仅表示当前子集筛查通过；manual_review 表示软件已明确要求人工复核；fail 表示按当前输入和规则子集未满足筛查条件，自动结论不得用于施工图。
