# V3.19 设计院专家式支撑—配筋—墙长联合设计

## 1. 适用目标

本模块将水平支撑拓扑、施工阶段、围护墙内力、钢筋笼配筋和围护墙竖向长度作为同一设计问题处理。系统用于方案设计、深化筛查和可编辑成果生成，正式施工图仍需结合完整勘察、水文地质、施工组织、节点计算和注册工程师审签。

## 2. 水平支撑专家流程

1. 根据基坑局部主轴、长宽比、凹凸性和施工分区选择支撑体系族。
2. 狭长基坑优先短跨直对撑；转角采用墙/围檩—墙/围檩直接角撑。
3. 普通支撑禁止跨中贯穿交叉；双向体系的次支撑只能在专用 T/Y 节点终止，并配置竖向承托。
4. 环撑、边桁架和正交刚架必须显式选型，并采用与构造一致的二维或三维整体模型。
5. 每个候选必须完成分阶段计算后才显示轴力、位移和方案排名；几何代理不得作为设计内力。

## 3. 围护墙双向配筋

### 3.1 沿深度分区

- 冠梁区；
- 支撑节点区；
- 坑底转换区；
- 普通受力区；
- 墙趾区。

坑内侧和坑外侧分别采用施工阶段正负弯矩包络设计。

### 3.2 沿平面分区

- 转角返回区；
- 支撑反力引入区；
- 长墙普通区；
- 墙幅接头和施工缝区。

支撑节点附加筋保留支撑编号、标高和墙面里程，禁止将局部附加筋错误铺满整面墙。

### 3.3 设计储备和防降级

- balanced 模式目标利用率默认不大于 0.88；
- conservative 模式目标利用率不大于 0.80；
- economic 模式目标利用率不大于 0.95；
- 优化器不得无提示削弱既有连续钢筋笼；
- 长墙基础笼默认最大主筋间距 150 mm，普通墙默认 180 mm；该值属于项目设计偏好，仍需裂缝和构造复核；
- 配筋同时满足承载力、净距、保护层、钢筋笼制作、连接、吊装和施工缝要求。

三维浏览器只进行受控采样。真实数量取物理墙长除以钢筋间距，完整数量保存在钢筋表、加工 CSV/JSON 和施工图中。

## 4. 围护墙长度优化

系统分别处理两类“墙长”：

1. 平面设计段和施工槽段长度：用于统一同一墙面的设计参数、墙幅分段、接头位置和钢筋笼吊装；
2. 竖向墙长/墙趾标高：用于嵌固、抗隆起、渗流、防水帷幕和经济性比较。

竖向候选包括统一墙趾、连续分区墙趾和保持现状。导入或人工锁定墙趾禁止自动缩短。分区墙趾须满足：

- 连续区段长度不小于项目下限；
- 相邻墙趾高差不超过项目限值；
- 台阶位于槽段边界、转角或明确施工缝；
- 重新完成全部施工阶段、嵌固、抗隆起、抗突涌、渗流和变形计算；
- 不削弱永久防水、主体结构和相邻环境控制要求。

## 5. 发行门禁

以下任一条件存在时仅允许输出审查版：

- 候选方案尚未完成完整分阶段计算；
- 普通支撑存在交叉或支撑—支撑非专用节点；
- 墙体配筋利用率、裂缝、锚固、搭接或净距未闭环；
- 钢筋几何被截断或加工包存在硬失败；
- 墙趾优化后未重新完成水文地质和稳定复核；
- 节点局部承压、加劲、施工时序或审签未完成。
