# 计算方法与工程边界

墙体分析采用分阶段侧压力与一维弹性地基梁有限差分模型。坑内开挖区不施加被动侧土弹簧，开挖面以下按分层土体参数形成水平地基反力。墙顶采用自由端边界，墙趾约束由嵌固段土弹簧和边界条件共同表达。

每面墙仅接收端点实际连接到该墙段的支撑。单根支撑水平等效刚度由构件截面、材料弹性模量、实际长度、与墙面法向夹角及构件角色计算。围檩连续梁和全局墙—围檩—支撑矩阵用于分配节点反力与支撑轴力。

稳定审查包含嵌固、抗隆起、承压水、渗流、整体稳定和软弱下卧层等子集。钢筋、围檩、支撑和立柱基础输出属于可追溯的初步设计或筛查结果。

当前边界包括：土体本构仍以工程简化参数为主；空间耦合模型仍属于杆系/弹簧代理；未替代经验证的三维有限元专项分析；规范实现属于条文子集，条文适用条件需专业复核；施工监测反演尚未形成生产闭环。
