# PitGuard V3.11.0：规范追溯、钢筋成套交付与在线计算文档

## P0

- 修复钢筋包下载为项目 JSON 的错误链接；
- 钢筋深化任务改为 ZIP 成套输出；
- 完整交付清单和 CAD 清单移除 100% 硬编码；
- Assurance 统一聚合总检查和阶段检查；
- 人工复核数量由实际检查结果生成。

## P1

- 新增项目级设计流程—规范矩阵；
- 八个工作流步骤显著显示主控规范和强制等级；
- 展示规则 ID、条文引用、计算状态和成果输出；
- DOCX 计算书加入规范矩阵和计算原理表；
- 钢筋 ZIP 提供 XLSX、CSV、JSON、检查表和使用说明。

## P2

- 在线文档扩展为操作、原理、规范和交付四个章节；
- 六类关键计算模块给出输入、模型、公式、假定、输出和验证点；
- 建立机器可读的规范目录和项目级追溯 API；
- 规范、计算书、工作台和交付包统一使用同一数据源。

## 数值求解质量门禁

V3.11.0 在墙—围檩—支撑全局矩阵中补充 `K·u=F` 残差、原始矩阵残差、矩阵对称误差、条件数和正则化状态。该门禁用于识别矩阵病态、约束不足和正则化求解，独立于工程规范验算。正则化求解自动进入人工复核，严重病态矩阵阻断正式采用。
