# PitGuard V3.14.0：丰收湖项目强度驱动计算恢复

## 版本目标

V3.14.0 针对丰收湖实际异形基坑出现的 36 项计算硬失败，完成计算模型纠偏、强度前置设计、自动拓扑恢复、施工阶段传力修正和诊断界面升级。

## 关键改动

- 短回墙和阶梯墙段建立可追溯的角部/围檩传力约束；
- 修复第一道支撑标高 0.0 m 被过滤的问题；
- 拆换撑阶段将已形成楼板和永久围檩纳入竖向压力分带；
- 围檩采用闭合外围多跨连续包络，并检查静力平衡；
- 支点间距和支撑有效无侧向支承长度前置为方案硬门禁；
- 自动增补角部扇形斜撑、局部短对撑和共享交叉节点；
- 次对撑/角撑之间的设计交叉必须具有明确共享立柱；
- 全局矩阵条件数诊断改为有界成本算法；
- 计算恢复面板显示修复前后拓扑、强度和刚度状态；
- 在线文档和 DOCX 计算书增加强度驱动设计记录。

## 实际项目结果

修复后：Pass 1444、Fail 0、Warning 5、Manual review 0；最大墙体位移 3.418 mm，最大墙体弯矩 657.737 kN·m/m，最大支撑轴力 5261.129 kN，强度、刚度和稳定性均通过。
