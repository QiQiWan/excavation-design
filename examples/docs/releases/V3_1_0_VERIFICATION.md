## 3. 样例项目验证

样例：`URBAN-TOPDOWN-32M-WALL-5SUPPORT`。

### 配筋方案

- 墙体分区：40 个；
- 钢筋混凝土支撑方案：110 个；
- 梁与节点方案：244 个；
- 检查项：434 个；
- 通过：386 个；
- 警告/复核：24 个；
- 失败：24 个；
- 计算包络异常：0 个；
- 需要升级墙体截面或配筋体系：24 个。

样例的 24 个失败项来自部分墙体分区的计算需求超过当前预设单层钢筋组合能力。系统保留失败状态并给出增大墙厚、经核验的多排钢筋或机械连接体系等建议，没有通过切换经济模式隐藏该风险。该结果同时说明当前规范算法样例仍需更高等级结构模型和项目级截面复核。

### CAD 图纸包

- 计划图纸：29 张；
- 实际 DXF：34 个；
- CSV 表格：17 个；

- V2.x 下游脚本依赖的根目录 CSV 兼容副本。

## 4. 已验证的一致性

- 应用方案后，每幅墙均回写控制钢筋组；
- 钢筋混凝土支撑均可回写纵筋、端部箍筋、跨中箍筋和附加筋；
- 三维墙体钢筋携带 `zoneId`、侧别、分区标高和图纸引用；
- 三维抽样覆盖所有墙段和支撑宿主，不再集中截取首批构件；
- CAD 图纸目录、实际文件、配筋方案和钢筋表均由同一项目数据生成；
- `full`、`general`、`rebar`、`details` 四类导出范围可以独立生成；
- 深基坑负高程立面标题栏依据图元边界自动放置。
