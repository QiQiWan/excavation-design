# V3.1.0 配筋与 CAD 成套出图迭代

本版本集中推进地下连续墙、钢筋混凝土支撑、围檩和节点的分区配筋、三维审查与施工图纸成套导出。

## 主要改动

### 分区配筋

- 新增 `conservative`、`balanced`、`economic` 三种配筋模式。
- 地下连续墙按支撑节点区、开挖转换区、墙趾区、墙顶区和一般区生成坑内/坑外侧配筋。
- 保存包络来源、控制内力、分区标高、计算需求、提供面积、推荐墙厚和施工性说明。
- 钢筋混凝土支撑区分端部加密区和跨中区，输出纵筋、箍筋、拉结筋及错开搭接方案。
- 围檩、冠梁、环梁和节点增加 U 形附加筋、约束箍筋及抗劈裂钢筋网建议。
- 应用方案后将控制钢筋回写至构件，同时保留完整分区方案作为 CAD 和三维显示主数据。

### 三维显示

- 墙体钢筋与分区标高、坑内/坑外侧和图纸编号关联。
- 增加按钢筋类型、状态和宿主构件着色。
- 增加三向剖切、宿主透明度、钢筋显示倍率、构造标注及单构件隔离。
- 采用宿主轮询采样，优先保留失败、警告和人工复核对象。
- 选中对象可查看分区、包络来源和关联图号。

### CAD 图纸包

- 图纸包按总说明、平面、剖面、配筋、大样和表格分目录组织。
- 新增围护—支撑总平面和逐层支撑平面。
- 新增墙体配筋总图、全部墙幅分区立面和逐幅独立分区立面。
- 新增支撑、围檩、冠梁和环梁配筋总图。
- 节点大样扩展至 D-01～D-07，补充墙幅接头、钢筋笼连接、支撑端部锚固和错开搭接。
- 标题栏依据图元边界自动放置，减少负高程立面与标题栏重叠。
- 支持 `full`、`general`、`rebar`、`details` 四种导出范围。
- 新增图纸目录、模型绑定、实际生成文件清单、配筋方案和质量表格。
- 保留 V2.x 平面 CSV 文件名兼容副本。

## 工程控制原则

- 配筋模式不会自动掩盖承载失败。
- 需求超过当前单层组合能力时输出失败状态和截面/配筋体系升级建议。
- 计算包络缺失或异常时标记来源和复算要求。
- 浏览器三维钢筋为可审查抽样；下料数量以逐根几何和钢筋表为准。
- DXF 为可编辑 R12 模型空间图纸底稿，正式签发前需按企业标准完成纸空间、图签、审签和项目专项构造。
