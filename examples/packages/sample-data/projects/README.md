# 示例项目

运行：

```bash
python scripts/seed_sample_project.py
```

会生成完整流程示例：

- `sample_full_project.json`：已完成地勘导入、地质模型、VTU、基坑、围护设计、施工工况、计算和校核的项目数据。
- `sample_full_project.ifc`：工程级 IFC4 STEP 示例模型，包含 IfcWall/IfcBeam/IfcColumn、材料和设计属性集。
- `sample_full_project_calculation_report.docx`：DOCX 计算书示例，包含设计依据、公式、校核结果、声明和结论。
- `sample_full_project_summary.json`：流程执行摘要。

该示例用于验证软件流程和接口完整性，不可作为真实工程施工图依据。
