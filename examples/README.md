# PitGuard V2.3.0 public-paper-derived benchmark cases

This package contains normative-algorithm regression cases derived from publicly described excavation projects. It intentionally avoids finite-element solvers. Each case includes project JSON, issue report, calculation trace, CAD/SVG drawings, DOCX report and construction-visual IFC. Geometry/depth may be normalized or capped for repeatable automated regression; source dimensions remain recorded in benchmark_summary.json.
