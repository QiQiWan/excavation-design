# V3.13.0 图纸体系与协同成果导出迭代

## 版本目标

本版本集中解决两类问题：单张 CAD 表达深度不足，以及不同成果文件分散、状态口径不统一、移交后难以验证完整性。

## 图纸迭代

- 新增 G-01、S-01、S-04、S-05、Q-05。
- 深化 S-03、R-01、R-02、R-04、R-05、D-03、D-06、D-07。
- 建立逐图实体、文字、尺寸、图层、图签、纸空间和视口质量检查。
- 新增图纸—模型—计算—规范矩阵。
- 增加旧项目图纸规则自动迁移，避免升级后缺失核心图种。

## 成果导出迭代

- 新增协同成果交付包接口和前端入口。
- 将图纸、计算书、IFC、钢筋深化、项目快照和审计文件统一组织。
- 提取高频审查文件，减少多层 ZIP 查找。
- 增加移交单、成果登记表、验收矩阵、关系矩阵、SHA256 和离线校验脚本。
- 发行等级统一为 `development_only`、`review_complete` 和 `construction_issued`。

## 样例验证

公开基准项目输出 41 张图纸，逐图表达质量结果为 41/41 通过，质量得分 100。该结果仅说明图纸文件层面的表达和可解析性通过；基准项目计算和审签仍存在失败项，因此协同交付包发行等级为 `development_only`。
