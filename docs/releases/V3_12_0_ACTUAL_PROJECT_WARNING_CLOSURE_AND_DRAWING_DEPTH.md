# V3.12.0 实际工程导入、预警闭环、施工图深化与规范矩阵重构

## 主要更新

- 增加 24 孔、144 层记录的实际工程 CSV 和 20 点异形基坑轮廓命令。
- 钻孔导入支持饱和重度、压缩模量和 X/Y/Z 三向渗透系数。
- 项目设置支持明确指定支撑深度；实际案例一键脚本按 0.0、4.0、7.2、10.3、13.3 m 生成五层支撑，并写回墙底标高 -32.8 m。
- 重复地层参数异常按风险合并，避免同一异常在每个钻孔重复显示。
- 修复最下一道支撑恰好位于阈值时产生伪预警的问题。
- 长支撑检查采用有效无侧向支承长度，已被立柱分段的构件不再重复警告。
- 自动端点吸附、候选删除、短跨优先和凹角拓扑等成功动作转入设计证据。
- 地连墙初选边界使用 `manual_review` 表达，不再作为普通算法 Warning。
- 重构规范矩阵：每个具体计算直接绑定规范、条文关注、规则证据、状态和输出。
- 深化支撑分层图、立柱图、D-06 墙幅接头止水与钢筋笼连接图、D-07 支撑锚固和施工缝图。
- 图纸完整性门禁增加关键详图、图层和实体深度检查。
