# PitGuard V3.76.0 验证矩阵与外部基准

V3.76 建立结构内核验证矩阵，避免以单一算例或单一残差作为正确性证明。

## 验证层级

- 单元级：空间梁刚度矩阵对称性、刚体模态和特征值检查；
- 规则级：钢支撑稳定曲线随有效长度单调变化；
- 独立实现：二维框架、闭合环梁—径向桁架及三向转动节点与独立 SciPy 组装求解对比；
- 成熟软件：运行环境存在 OpenSeesPy 时执行外部基准，缺失时明确标记 unavailable，不生成等效性声明。

正式发行可配置为必须具有当前结构内核版本对应的外部基准证书。验证矩阵不覆盖土体非线性、固结、接触、动力和现场施工偏差。
