# PitGuard V3.87.10 安装与验收

## 安装

1. 停止现有服务。
2. 备份 `services/api/pitguard.sqlite3`。
3. 用 V3.87.10 完整包替换代码目录。
4. 恢复数据库并启动 `./start-dev.sh`。
5. 打开原项目，在“配筋深化”执行一次主配筋闭合操作，使历史槽段几何持久化为当前墙路径。

## 验收 A：平面轮廓

1. 在工程输入/围护方案页面记录基坑轮廓和围护墙折点。
2. 打开配筋三维视图并切换到俯视。
3. 钢筋笼中心线、围护墙宿主轮廓与当前围护墙轴线应逐段重合。
4. 凹角、凸角和局部折返处不得再被首尾弦线切割。

## 验收 B：槽段连续性

1. 查看“围护墙钢筋覆盖/几何审计”。
2. 应显示全部墙面已解析。
3. 槽段沿墙轴线连续，无明显间隙、重叠或跨越基坑内部的直线。
4. 旧项目首次打开可能显示“已自动修复”，重新应用配筋后应持久化。

## 验收 C：导出一致性

1. 导出 IFC、CAD 和槽段追溯表。
2. 三类成果中的槽段编号、长度与平面路径应一致。
3. 槽段总长度与当前围护墙轴线长度在容差内一致。

## 验收 D：回归

- 水平支撑纵筋、箍筋、构造筋和拉结筋仍可显示；
- 围护墙宿主数量与钢筋笼覆盖数量一致；
- 计算合同和配筋合同不应因纯几何自修复被误判为缺失。
