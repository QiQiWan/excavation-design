# V3.87.9 安装与验收

## 安装

1. 停止旧服务并备份 `services/api/pitguard.sqlite3`。
2. 解压完整包并恢复数据库。
3. 执行 `chmod +x start-dev.sh start-linux-dev.sh start-macos-dev.sh stop-dev.sh scripts/*.sh`。
4. 执行 `./start-dev.sh`。

## 验收场景

### 场景 A：计算闭合但配筋入口显示合同失效

1. 打开原工程的配筋深化页。
2. 点击“后台闭合配筋”或主配筋闭合操作。
3. 页面应显示“正在自动恢复计算合同……”处理中间状态。
4. 任务完成后，`CALCULATION_NOT_CURRENT` 应自动消失。
5. P3 入口应继续执行或仅保留真实专业复核项。

### 场景 B：设计快照确实发生变化

1. 修改墙厚、支撑拓扑或正式施工工况。
2. 配筋入口应列出具体合同差异字段。
3. 主操作应先完成当前方案复算，再重新生成配筋。
4. 新配筋合同中的计算结果 ID、设计快照哈希和支撑拓扑哈希应与当前工程一致。

### 场景 C：真实硬失败

当结构计算仍有硬失败时，系统必须保留阻断，不得以自动恢复状态掩盖。

## 日志检查

任务日志应出现：

- 权威施工阶段结果已装载；
- 自动恢复计算合同；
- 重新生成配筋；
- 从已保存工程重建最终配筋合同。
