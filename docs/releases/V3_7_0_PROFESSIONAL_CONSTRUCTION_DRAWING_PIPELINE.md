# V3.7.0 专业施工图生产链迭代

V3.7.0 按四个阶段重构出图链路：可靠 CAD 内核与纸空间出版、结构图形与原生标注、钢筋加工深化与几何净距、正式发行质量门禁。

## 阶段一：CAD 内核与纸空间出版

- 手写 R12 DXF 输出替换为 `ezdxf` R2018（AC1032）内核。
- 模型空间统一采用毫米、1:1 建模；既有生成器继续接受米制坐标，由写入器统一转换。
- 每张图建立 A0/A1/A2/A3/A4 纸空间布局、锁定视口、图框和图签。
- 增加 `PIT_CN` 中文文字样式、`PIT_DIM` 原生标注样式、图层表和标题栏块。
- CAD 包自动执行独立解析、版本、单位、纸空间、视口和 DXF audit 校验。
- 新增 `packages/cad-standards/default-publish-profile.json` 与 `scripts/validate-cad-package.py`。
- 正式 PDF 改为 ReportLab 矢量多页出版，一张登记图对应一页，中文采用 CID 字体映射。

## 阶段二：结构图形与标注

- 围护墙、围檩和支撑由单线升级为真实宽度轮廓加中心线。
- 偏移支撑端点与墙面连接点之间绘制短刚臂，图面与计算连接关系一致。
- 平面、分层平面、控制剖面、墙体配筋立面和节点大样进入统一图纸清单。
- 尺寸采用原生 `DIMENSION` 实体，图号、比例、阶段、设计/校核/审定栏进入纸空间。
- 图纸完整性门禁检查总说明、总平面、控制剖面、墙体/支撑/围檩配筋图、节点大样和加工表。

## 阶段三：钢筋加工深化

- 逐根钢筋按 12 m 定尺和运输限制自动分段。
- D25 及以上、地连墙纵筋优先采用机械连接；其余按搭接长度生成接头。
- 生成加工段、接头错开组、套筒规格、下料长度和重量。
- 生成逐根几何净距检查，区分中心距、净距和最低要求。
- 修复动态钢筋组重复编号，输出确定性唯一钢筋 ID。
- CAD 包新增完整加工表、加工段表和几何净距检查表。

## 阶段四：正式发行与质量门禁

- 施工版同时检查有效计算、无工程硬失败、无配筋/加工硬失败、DXF 合法、图纸完整性、四级审签和当前修订绑定。
- 审查版允许生成，但始终保留红色审查水印和门禁报告。
- CAD 包输出 `dxf_validation.json`、`drawing_completeness.json`、`construction_issue_gate.json` 和 `SHA256SUMS.txt`。
- 正式包输出 CAD、逐页 PDF、规则集、图纸计划、审签台账、修订记录和 DWG 转换说明。

## 当前边界

- 原生 DWG 转换和企业 CTB/STB 仍需 AutoCAD、BricsCAD 或 ODA 运行环境。
- 预埋件、声测管、导管和复杂机械接头需要项目提供真实几何后，才能给出最终施工级碰撞结论。
- 当前逐根几何有浏览器/内存上限；加工表可生成，但省略的逐根几何会阻断施工版完整性门禁。
- 注册工程师签章、现场条件复核和企业图审仍属于正式发行必要环节。
