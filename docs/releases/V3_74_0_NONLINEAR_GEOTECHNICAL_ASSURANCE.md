# PitGuard V3.74.0 非线性岩土与地下水保证

V3.74 增加水平地基反力参数溯源、位移动员诊断、切线刚度退化和水位敏感性分析。

## 计算链

1. 优先读取土层明确给出的水平地基反力系数；
2. 缺失时允许在初步设计中依据弹性模量透明推导；
3. 仍缺失时仅允许使用明确标记的筛查默认值；
4. 按墙体位移计算土体抗力动员比例和切线刚度比例；
5. 自动生成低/高土体刚度及水位上升 0.5 m、1.0 m 的敏感性情景。

## 正式门禁

经验推导和固定默认值不具备正式使用资格。正式模式需要经核验的土层参数、钻孔、水位记录和项目确认的分析等级。当前模块用于非线性风险诊断，尚未替代二维/三维弹塑性土体有限元、接触、固结和渗流耦合。
