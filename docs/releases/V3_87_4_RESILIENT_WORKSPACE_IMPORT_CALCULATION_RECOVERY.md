# PitGuard V3.87.4 稳健工作台、异步地勘导入与计算阻断闭环

## 修复范围

1. 设计基准保存后不再返回项目列表。工程选择由 `/projects/:id` 路由持久化；保存响应不完整时重新读取规范化工程对象，并恢复当前阶段与滚动位置。
2. 钻孔 CSV/XLSX/XLSM 上传采用流式暂存，解析、校验、原始文件证据保存和项目版本写入均在独立 worker 中执行。前端显示任务进度、心跳和取消入口；解析器限制异常行列范围、关闭工作簿、清理陈旧暂存文件，并配置独立硬超时，避免单个异常表格拖死交互服务。
3. 完整计算在正式求解前自动执行有界修复：闭合轮廓墙段映射、缺失基础支撑、地质设计域覆盖、墙/围檩支承语义和支撑候选拓扑。修复后重新执行资格门禁。
4. 新增“自动诊断、修复并复算”任务。前置条件关闭后强制运行验算—优化—再验算闭环；第一阶段仍存在可安全处理的结构问题时，系统自动应用有限数量的墙趾加深、局部截面增强或支撑深化措施，并进行第二阶段复算。测量坐标、土体参数、水位和锁定施工顺序仍保留人工确认。

## 工程边界

- 系统不会自动执行测量坐标平移、旋转或 m/mm 单位转换。
- 自交轮廓、点数不足、人工专项转接体系和无合格候选仍需工程师处理。
- 自动修复会保存项目版本、失效旧计算证据并保留恢复历史。

## 验收

- 保存设计基准后 URL、工程和当前“设计基准”页面保持不变。
- 导入大文件时主页面持续响应，任务状态可见，导入成功后钻孔和地层刷新。
- 普通计算和自动修复计算均先运行资格恢复；可自动关闭的计算后超限继续进入第二阶段加固复算，剩余阻断在计算页显示具体原因、自动动作和人工动作。
