# PitGuard V3.42.0 专业可视化与设计校核恢复

## 1. 目标

V3.41 将核心工作台收敛为五阶段流程，但工程可视化和设计校核被压缩为少量 SVG 摘要，不能满足设计人员对三维模型、构件级设计值、施工阶段内力包络和配筋明细的复核需求。

V3.42 恢复 V3.39 以前的专业可视化深度，同时维持 V3.40 之后的核心产品边界：不恢复工业成熟度、监测、问题中心、基准算例和高级套件等外围模块。

## 2. 工作台模式

五阶段工作台增加两种显示深度：

- `专业视图`：默认模式，加载对应阶段的完整工程可视化和设计校核模块；
- `精简视图`：保留 V3.41 的轻量摘要和 SVG 快速预览。

模式切换仅影响展示层，不改变项目数据、计算结果和设计结论。

## 3. 恢复的专业模块

### 3.1 工程输入

专业视图恢复：

- 地质三维模型；
- 钻孔、地层界面和地下水位；
- 地质曲面与 VTU 网格摘要；
- 地质模型警告和字段映射。

完整曲面继续使用外部对象和预览曲面，网页不读取完整 IDW 数据。

### 3.2 围护方案

专业视图恢复：

- A/B/C 大幅面方案预览；
- 方案缩放、平移和构件编号；
- 支撑、立柱、角撑和墙上/内部汇交节点；
- 完整方案比选表；
- 围护结构三维模型；
- 支撑布置评分平面；
- 围护墙、围檩、支撑和立柱构件明细。

候选几何仍通过轻量预览缓存读取，方案切换不加载完整项目快照。

### 3.3 计算验算

专业视图恢复：

- 最大合成侧向压力、支撑轴力、墙体弯矩、剪力和位移设计值；
- 围护墙沿深度的弯矩、剪力和变形包络；
- 围檩正负弯矩、剪力和挠度包络；
- 支撑轴力柱状分布；
- 墙体平面控制云图和墙身深度云图；
- 三维墙体内力/变形云图；
- 强度、刚度和稳定性汇总；
- 围檩连续梁节点反力；
- 墙—围檩—支撑全局矩阵和换撑刚度；
- 地下水、抗隆起、突涌、渗流和整体稳定专项结果；
- 构件级计算值、规范限值、状态和结论表。

施工阶段数组仍按分片按需加载，打开项目不会一次读取全部阶段结果。

### 3.4 配筋深化

专业视图恢复：

- 配筋策略切换；
- 墙体分区和立面配筋；
- 支撑纵筋和截面优化；
- 围檩、节点附加筋和承压利用率；
- 计算有效性、支撑传力体系和图纸闸门；
- 钢筋施工图目录；
- 逐根钢筋三维检查。

逐根钢筋三维数据采用显式按钮加载，不随配筋页面自动请求。

## 4. 性能边界

专业模块全部采用动态导入。生产构建拆分为独立文件：

- GeologyViewer：约 1.87 kB；
- RetainingSystemViewer：约 15.62 kB；
- SchemeComparisonPanel：约 16.93 kB；
- RebarDesignPanel：约 22.19 kB；
- ResultViewer：约 60.73 kB；
- Three.js：约 523.86 kB，仅进入需要三维视图的异步块。

Three.js 不进入项目列表和普通首屏主包。配筋逐根钢筋可视化继续按需触发，避免恢复旧版“进入页面即加载全部重型数据”的问题。

## 5. Core 模式兼容

专业计算结果模块新增 `coreMode`。在 Core 模式下保留内力、变形、稳定性、构件校核和方案比选，隐藏需要 Full 路由的专家墙长优化和冗余评估面板，避免默认产品调用未注册的高级接口。

## 7. 设计原则

V3.42 恢复的是设计人员需要的工程证据和交互深度。外围治理模块仍保持移除，项目存储结构和后台任务边界不回退。
