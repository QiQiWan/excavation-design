# V3.87.11 验收

1. 依次打开地质、围护、计算云图和配筋三维模型，确认右上角均显示“全屏”。
2. 进入全屏后模型自动铺满可用区域，退出后恢复原尺寸。
3. Safari 禁止原生全屏时应自动进入页面内全屏，Esc 可退出。
4. 连续切换 20 次工作阶段和三维模型，不应出现 `Error creating WebGL context`；发生上下文丢失时只显示当前模型重建按钮。
5. 关闭浏览器标签页再返回，后台任务进度保持单调且不重复提交。
6. 停止 worker 后顶部显示“流程降级/阻断”；恢复 worker 后在下一次健康轮询自动恢复。
7. 快速连续保存两次不同参数，旧刷新响应不得覆盖最新项目。
