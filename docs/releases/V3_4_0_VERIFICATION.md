## 专项覆盖

- 核心和企业规则包发现、同名覆盖及稳定来源标识；
- 安全条件 DSL、最大嵌套深度和上下文根限制；
- 未知配置结构版本、未知渲染器和不安全路径拒绝；
- 总图、配筋、节点大样、质量复核和监测反演模块启停；
- 必需图纸模块及最低工程图纸能力不可关闭；
- 按支撑层和按墙幅动态展开；
- 每张图合并 1、2 或多幅墙体立面；
- 图幅、方向、有效图幅比例和允许比例自动适配；
- 项目自定义规则保留式优化与标准预设对照；
- 优化候选元数据轻量返回及服务端重建采用；
- 施工版发行规则只能加严，不能绕过工程闸门；
- CAD 包包含规则集、图纸计划和逐规则决策轨迹；
- 正式发行包和 PDF 发布清单记录规则集哈希与计划哈希。

- 单墙/组合墙配筋立面：2 张；

- 图纸计划哈希：`1916d632401050c0`；
- 规则优化候选：30 个；
- 优化响应采用元数据模式，约 39 KB；
- CAD 审查包约 544 KB；
- 正式图纸审查包约 698 KB；

优化器在当前权重下推荐 A2 施工深化型、每张图合并四幅墙体立面，评分 96.01。该推荐用于图纸组织决策，施工版发行仍受工程计算、配筋、审签和修订闸门控制。

## 已知边界

- 新图种仍需开发并注册服务端白名单渲染器；规则包只能组合已有绘图能力。
- JSON Schema 文件用于外部编辑器提示，服务端最终以安全校验器结果为准。
- 当前优化采用离散候选枚举，尚未引入历史项目学习或贝叶斯优化。
- Three.js 公共包约 523.86 KB，仍高于默认 500 KB 提示线，与本轮出图规则功能无直接耦合。
