# PitGuard V3.77.0 危大工程法定流程、监测与在线文档

V3.77 将工程计算门禁扩展为危大工程全过程证据门禁，并同步更新在线文档。

## 法定流程台账

系统记录并校验：

- 真实、准确、完整的地质、水文地质和周边环境资料；
- 危大工程清单和分类确认；
- 专项施工方案及施工单位审核；
- 监理审查；
- 超过一定规模工程的专家论证及整改闭环；
- 监测方案、技术交底、应急预案；
- 分阶段验收和档案清单；
- 注册结构工程师复核与批准状态。

每条证据绑定文件标识、SHA-256、核验人、项目快照和支撑拓扑。软件自动筛查只产生提醒，项目分类必须由责任主体确认。

## 在线文档

`/api/documentation` 新增：

- L0–L3 分析等级；
- 合规值来源语义；
- V3.73–V3.77 路线；
- 危大工程证据链；
- 准确度、验证与正式发行边界。

前端在线文档增加“准确度与验证”和“合规与法定流程”两个章节。
