# PitGuard V3.73.0 准确度分级、参数来源与合规语义

V3.73 将“计算得到结果”和“结果可以用于正式设计”拆分。系统对几何拓扑、土水压力、墙土相互作用、平面支撑体系、空间结构、岩土稳定、构件设计和监测反馈分别标注 L0–L3 分析等级。

## 核心能力

- L0：算法筛查；L1：规范简化计算；L2：验证型工程计算；L3：高级专项复核。
- 记录土层参数来源、置信度、缺失项、已核验钻孔和地下水记录。
- 识别默认地基反力、经验推导、凝聚模型、代理模型和数值回退。
- 区分规范控制值、企业标准值、项目控制值、算法建议值和模板默认值。
- 正式模式下，默认参数、代理/回退模型、低于项目要求的控制分析域自动阻断。

## 当前边界

分析等级用于表达模型保真度和验证深度，不能替代工程师对适用条件和规范条文的判断。朗肯/Jaky 土压力、简化稳定分析即使达到输入完整，也仍需在复杂工程中采用 L3 专项复核。
