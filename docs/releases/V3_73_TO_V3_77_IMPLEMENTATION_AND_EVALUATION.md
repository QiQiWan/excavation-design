# PitGuard V3.73–V3.77 深化实施、验证与交付报告

## 1. 发行结论

本次从 V3.72.0 按依赖顺序推进至 V3.77.0，代码、后端在线文档、前端在线文档页面、版本说明、验证脚本和量化评估数据已同步更新。

本轮形成五条相互独立的保证链：

1. **准确度语义链**：分析等级、参数来源、默认值、代理模型和回退路径可追溯；
2. **岩土模型链**：地基反力来源、位移动员、切线刚度及土体刚度和水位敏感性可见；
3. **空间结构链**：异形转接体系增加六自由度线弹性空间验证，钢支撑稳定采用可追溯屈曲曲线；
4. **验证链**：单元、组装、独立参考模型和外部结构软件证书分层记录；
5. **法定流程链**：危大工程分类、专项方案、监理审查、专家论证、监测、交底、应急、验收、归档和注册工程师审批纳入发行门禁。

当前版本能够安全地阻断缺少真实地质、水位、法定证据和外部验证的合成项目。它没有把数值收敛、结果数量或几何闭合解释为正式设计完成。

## 3. V3.73：准确度与合规语义

### 3.1 分析等级

系统为每个控制域记录独立等级：

- **L0**：算法几何或经验筛查；
- **L1**：规范简化计算或工程代理模型；
- **L2**：具有输入溯源、施工阶段、数值保证和基准验证的设计计算；
- **L3**：经高级有限元、成熟软件或真实工程校准的复核结果。

正式模式可配置最低等级。低于项目要求的控制域直接进入阻断清单。

### 3.2 参数和限值来源

新增来源分类：

- 规范控制值；
- 企业控制值；
- 项目批准值；
- 算法建议值；
- 待确认默认值。

默认值、经验回退和代理模型均记录使用对象、原因、影响域和正式使用资格。软件内部建议值不会继续伪装成规范限值。

### 3.3 条文追溯

计算检查汇总增加标准编号、条文引用、状态统计和代表性证据。当前实现用于追溯已编码规则，不能据此宣称某部标准已经全文自动化。

## 4. V3.74：岩土与地下水保证

### 4.1 地基反力优先级

水平地基反力采用以下优先级：

1. 经核验地层的显式水平基床系数；
2. 由土体参数推导的经验值，仅用于初步设计；
3. 固定默认值，仅用于筛查，正式模式禁止。

每个阶段—墙段保存地基反力数值、来源、正式使用资格和分层值。

### 4.2 位移动员和切线刚度

诊断层根据墙体位移估计土体抗力动员程度和切线刚度退化，输出最大动员比、最小切线刚度比及逐深度点结果。

### 4.3 不确定性工况

默认生成四个有界敏感性情景：

- 土体刚度降低；
- 土体刚度提高；
- 水位上升 0.5 m；
- 水位上升 1.0 m。

这些情景用于风险排序和补充计算提示。正式工程需要按真实参数重新运行，不能直接采用投影值出图。

## 5. V3.75：六自由度空间验证与钢支撑稳定

### 5.1 空间模型

每个空间节点使用六个自由度：

\[
[u, v, w, \theta_x, \theta_y, \theta_z]^T
\]

模型包含：

- 三维 Euler–Bernoulli 梁单元；
- 三维桁架支撑；
- 环梁和转接梁；
- 支撑作用点处的梁段切分；
- 刚域长度；
- 半刚性刚度折减；
- 立柱竖向约束；
- 轴力、双向弯矩、剪力和扭矩恢复。

空间模型与二维平面框架比较位移和弯矩差异。超过项目容许差异时进入警告或阻断。

### 5.2 数值保证

六自由度模型复用刚度尺度化、矩阵秩、正定性、条件数分级和平衡残差门禁。模型断连、秩亏或病态时不允许通过正则化掩盖。

### 5.3 钢支撑稳定

钢管支撑从固定稳定系数升级为 Perry 型屈曲曲线，稳定折减随长细比、截面、材料和曲线类型变化。计算结果保留曲线、无量纲长细比和稳定折减系数。

## 6. V3.76：验证矩阵

验证矩阵分四层：

1. 单元级矩阵对称性和刚体模态；
2. 规则和构件公式性质测试；
3. 独立 SciPy 参考模型交叉验证；
4. OpenSeesPy 外部结构软件对比。

外部软件不可用时返回 `unavailable`，正式外部基准状态保持关闭。证书与结构内核版本绑定，旧版本证书不能覆盖新内核。

## 7. V3.77：危大工程流程与在线文档

3. 专项施工方案及施工单位审核；

5. 超过一定规模工程的专家论证与整改闭环；

8. 应急预案；
9. 分阶段验收；

证据必须绑定不可变文件对象和 SHA-256。仅提交文件编号或客户端 `verified=true` 不会被采信。

## 8. 量化评估

评估项目：L 形正交凹基坑，外包约 60 m × 45 m，开挖深度 16.0 m，7 个施工阶段，中心汇交框架转接体系。

| 指标 | 结果 |
|---|---:|
| 计算耗时 | 13.094 s |
| 支撑数量 | 72 |
| 转接梁数量 | 30 |
| 立柱数量 | 22 |
| 主计算最大尺度化条件数 | 1850.890 |
| 主计算最大相对残差 | 5.246e-14 |
| 六自由度尺度化条件数 | 943699.557 |
| 六自由度相对残差 | 7.001e-12 |
| 二维/六自由度最大相对差异 | 34.95% |
| 不确定性情景 | 4 |
| 分析域 | 8 |
| 结果域 | 18 |
| 法定证据要求 | 10 |
| 工程结果完整度 | 74.4% |
| 工程就绪度 | 50.0% |
| 正式发行就绪度 | 49.0% |

### 8.1 评估解释

- 主计算和六自由度模型均获得很小的平衡残差，说明已定义线性方程组求解一致；
- 二维/六自由度最大相对差异为 34.95%，超过当前 25% 容许值，因此空间验证状态为 `warning`；
- 合成项目没有真实钻孔、地层、水位核验、法定流程证据和注册工程师批准；
- OpenSeesPy 在本环境中不可用，外部基准状态为 `unavailable`；
- 系统将工程状态和交付状态均保持为 `fail`，事务状态为 `committed`，说明计算结果成功保存，同时没有错误放行正式成果。

当前主要阻断类别：

- `analysis_fidelity`
- `calculation_assurance`
- `calculation_check`
- `geotechnical_model`
- `shape_transfer_detailing`
- `statutory_workflow`
- `verification_matrix`

- 规则与异形基坑方案生成和拓扑筛查；
- 施工阶段墙—围檩—支撑计算；
- 异形转接平面框架及六自由度线弹性验证；
- 分析等级、参数来源、回退路径和结果完整度审查；

- 危大工程证据缺项管理；

- 真实且经核验的勘察、水位、周边环境和施工阶段资料；
- 项目适用规范和地方要求确认；
- 高级岩土/结构软件复核或 L3 证书；
- 专项施工方案、监理审查和必要的专家论证；
- 监测和分阶段验收资料；
- 注册工程师复核、批准和可信电子签署。
