# PitGuard V3.75.0 全局六自由度空间验证与钢支撑稳定

V3.75 建立线弹性空间杆系验证内核。每个节点采用六自由度：

\[
\mathbf d_i=[u_i,v_i,w_i,\theta_x,\theta_y,\theta_z]^T
\]

空间梁单元覆盖轴向、双向弯曲和 Saint-Venant 扭转；支撑采用空间桁架单元。模型纳入刚域有效长度和半刚性刚度折减，并与平面转接框架的最大位移和构件弯矩进行差异审计。

钢管支撑稳定验算由固定折减系数升级为可追溯的 Perry 型稳定曲线，输出截面面积、惯性矩、回转半径、有效长度、长细比、欧拉应力、归一化长细比、曲线类别和稳定折减系数。

当前空间模型仍为线弹性验证层。材料开裂、支撑初始缺陷、二阶效应、接触和空间土体作用需 L3 分析。
