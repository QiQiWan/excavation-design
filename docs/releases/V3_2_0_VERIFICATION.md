## 6. 已知边界

- 当前正交网格主要适用于规则或近规则深大长方形基坑，复杂多边形、圆形、偏心支撑和分区不连续开挖仍需专项拓扑优化。
- 双筋受弯分支用于快速设计与失效分类，正式设计仍需进一步考虑裂缝宽度、长期效应、钢筋应变协调、抗震构造和节点局部有限元。
- 32 项警告/复核主要属于构造和专业审查项。它们不构成自动阻断，但必须进入设计、校核、审核和注册工程师审查流程。
- Three.js 独立包略高于默认 500 kB 提示线，后续可按查看器拆分或延迟加载钢筋/云图模块。
- 软件施工图复核包仍属于设计辅助成果，正式盖章出图需要项目专项规范复核、企业图签和签审流程。
