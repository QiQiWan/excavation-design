# V1.9.0 墙-围檩-支撑全局联立刚度模型与专项复核说明

## 1. 目标

V1.9.0 将 V1.8 的“墙体、围檩、支撑结果汇总”升级为“统一刚度矩阵原型”。每个施工阶段、每个墙面切片均建立一个全局矩阵，统一表达墙体节点水平位移、围檩节点水平位移、支撑轴向弹簧、立柱竖向支承代理和施工阶段激活/失活。

## 2. 自由度体系

- `wall:h:<face>:<i>`：墙体沿深度方向离散节点的水平位移。
- `wale:h:<face>:<support>:<endpoint>`：支撑端部所在围檩节点的水平位移。
- 支撑杆轴向变形通过 `EA/L` 投影为围檩节点弹簧。
- 立柱竖向支承在二维切片模型中输出为 `columnVerticalSupports` 代理反力，用于立柱桩和临时立柱基础复核。
- 施工阶段激活/失活由 `ConstructionStage.active_support_ids` 和 `deactivated_support_ids` 控制。

## 3. 计算链路

```text
土压力/水压力剖面
→ 墙体节点等效水平荷载
→ 墙体梁平移刚度 + 被动区土弹簧
→ 围檩节点水平自由度
→ 支撑 EA/L 弹簧刚度
→ 全局 K u = F
→ 墙体位移、围檩节点位移、支撑反力、轴向变形
→ 强度/刚度/稳定性复核汇总
```

## 4. 计算书图表化

`services/api/app/reports/charts.py` 从 `CalculationResult` 已有数据生成 PNG 图表，并插入 DOCX：

- 墙体土压力图；
- 墙体位移图；
- 墙体弯矩图；
- 墙体剪力图；
- 围檩弯矩包络图；
- 围檩剪力包络图；
- 支撑轴力柱状图；
- 校核结果汇总图。

## 5. CAD 几何内核增强

前端 `ExcavationEditor` 新增：

- 闭合多段线几何 offset；
- 支护轴线/地下室外墙线由偏移线生成；
- 倒角、圆角、自交修复；
- `FILLET`、`CHAMFER`、`REPAIR` 命令；
- 保留 DXF 导入/导出、图层、障碍物绘制和坐标命令。

## 6. 地下水与稳定性专项

新增筛查项：

- `JGJ120-2012-DEWATERING-STAGE-SUBSET`：降水后坑内外水位差与嵌固/帷幕深度关系；
- `JGJ120-2012-LAYERED-SEEPAGE-GRADIENT-SUBSET`：按分层渗透系数识别高渗透风险；
- `JGJ120-2012-WEAK-UNDERLYING-LAYER-SUBSET`：坑底以下软弱下卧层快速识别；
- 继续保留承压水突涌、抗隆起、抗渗流和整体稳定圆弧搜索。

## 7. 设计复核体系

`DesignReviewSummary` 对所有检查项按强度、刚度、稳定性归类，输出：

- `strengthStatus`；
- `stiffnessStatus`；
- `stabilityStatus`；
- fail/warning 计数；
- 强度利用率、刚度利用率、最小稳定安全系数。

## 8. 当前边界

V1.9.0 仍为工程辅助设计原型。全局矩阵采用二维切片和凝聚平动自由度，尚未形成完整三维空间杆系/有限元内核。正式施工图仍需注册岩土/结构工程师复核规范适用性、构造、节点、监测和施工组织。
