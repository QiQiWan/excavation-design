# 项目删除、水平支撑体系与地质设计域闭环

## 1. 适用范围

本章说明 V3.17.0 对项目生命周期、水平内支撑方案生成和地质设计域检查的统一约束。内容适用于单一闭合基坑轮廓的方案设计、初步设计和施工图深化辅助。多基坑、带内岛、环撑、中心岛、盖挖、桁架及明显三维高差体系应启用专门拓扑和人工复核。

## 2. 项目删除

项目列表提供“删除”操作。删除属于不可逆动作，前端必须显示项目名称并要求二次确认。后端按以下顺序执行：

1. 取消并删除该项目尚在内存中的任务；
2. 删除持久化任务记录；
3. 删除由任务管理器生成且位于配置导出根目录内的成果文件；
4. 删除项目数据库记录；
5. 清除项目级计算锁。

系统不会根据项目 JSON 中的任意路径删除外部文件，避免误删用户自行保存的勘察资料、企业模板和外部模型。

## 3. 常用水平支撑体系及算法表达

### 3.1 直对撑

直对撑的两端分别连接相对墙面或围檩，形成直接压杆传力路径。系统优先沿基坑局部短跨方向布置，并按围檩支点间距、支撑轴压稳定、施工净空和临时立柱服务跨共同确定间距。

### 3.2 转角角撑

角撑用于转角影响区，由一条墙面/围檩直接连接相邻墙面/围檩。V3.17.0 采用以下硬约束：

- 两个端点必须属于不同的相邻墙面；
- 不允许截断后抵在另一根普通水平支撑上；
- 墙面连接点距角点通常控制在 3.5～8.0 m，并不超过相邻墙长的 30%；
- 净距退让后的支撑中心线仍须位于坑内；
- 与普通支撑 proper crossing 数量必须为 0；
- 节点局部承压、围檩负弯矩和角部构造仍须深化验算。

### 3.3 八字撑、斜向分支撑

八字撑属于“主节点—墙面”或“主构件节点—墙面”的独立支撑族。其主节点必须具有明确的节点板、承压构造和竖向承托。系统不得将这类构件标记为角撑，也不得用它替代转角附近的墙—墙角撑。

### 3.4 双向网格与 T/Y 节点

对近方形、宽大或局部加宽基坑，可采用双向支撑。普通杆件不允许跨中穿越；次向支撑沿墙面法向连接到主支撑的明确 T/Y 节点，并在节点处终止。节点应设置临时立柱或经专项设计确认的竖向承托。

### 3.5 环撑—径向撑

只有项目明确选择闭合环梁或多边形环撑体系时，径向支撑才允许连接环梁。该体系需要单独计算环梁闭合受力、径向节点、局部承压、竖向支承和施工顺序，不能由普通对撑规则自动推断。

### 3.6 斜抛撑、中心岛和盖挖体系

斜抛撑、中心岛、桁架及盖挖逆作属于施工组织和空间体系高度耦合的特殊方案。系统只在输入完整的底部反力点、地下结构、施工阶段和障碍物后生成候选，否则保持人工设计边界。

## 4. 水平支撑方案生成流程

系统按照以下顺序生成和验收支撑：

1. 识别局部主轴、长短跨、凹角、凸角和局部加宽区；
2. 生成短跨直对撑；
3. 在转角影响区生成墙—墙角撑；
4. 对未满足围檩支点间距的墙段增加直对撑或墙法向 T/Y 分支；
5. 重建墙端、围檩端、T/Y 节点和临时立柱语义；
6. 删除坑外、悬空、过短、重复或普通同层交叉构件；
7. 同步施工阶段构件 ID；
8. 执行墙、围檩、支撑、节点、稳定和施工空间验算；
9. 将算法动作、尚未闭环风险和人工复核项分别记录。

方案评分不得替代硬约束。任一候选存在普通支撑穿越、角撑无双墙端、支撑中心线离开坑内、施工阶段引用失效构件或地质设计域未覆盖时，候选不得进入正式计算排名。

## 5. 地质设计域覆盖

### 5.1 两类状态

地质检查拆分为：

- `coverageStatus`：地质模型的实际平面范围是否覆盖基坑、围护墙、支撑墙面连接点、临时立柱和施工影响缓冲区；
- `extrapolationStatus`：覆盖过程中是否采用自动外扩或超出钻孔可信域的外推，以及外推距离是否可接受。

只有 `coverageStatus=fail` 才形成设计域硬阻断。模型已经覆盖设计域时，历史遗留的 `status=fail` 不得继续沿用。

### 5.2 自动外扩

当钻孔或原始地质网格范围小于所需设计域时，系统扩展网格到：

`基坑轮廓 ∪ 围护结构 ∪ 支撑墙面连接点 ∪ 临时立柱 ∪ 施工影响缓冲区`

施工影响缓冲区默认取 `max(10 m, 0.5 × 开挖深度)`。可信域外采用最近可信边界约束的插值延拓，并记录最大外推距离、方法和低置信度区域。自动外扩用于保证计算几何完整，不能代替补充勘察。

## 6. 丰收湖项目验证

V3.17.0 重新生成的基线体系包含 235 根支撑和 43 根临时立柱，其中 20 根角撑均直接连接相邻墙面/围檩，墙面连接点距角点约 3.8～4.1 m。普通同层穿越为 0，地质模型覆盖所需设计域且未使用水平外推。完整计算结果为 Fail 0、Warning 5，强度、刚度和稳定性均通过。

剩余 Warning 主要涉及首道支撑定位、预加轴力与温度、节点偏心、拆换撑施工时序以及坡道/出土口/邻近设施数据，仍需结合施工组织和监测方案闭环。
