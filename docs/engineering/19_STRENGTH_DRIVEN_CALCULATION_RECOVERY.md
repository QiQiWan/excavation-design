# 强度驱动的基坑方案—构件联合设计与计算恢复

## 1. 适用问题

当计算结果出现墙体受弯、受剪、位移、围檩承载力或支撑轴力硬失败时，系统不得仅在计算结束后放大构件截面。V3.14 将承载力、刚度和传力路径检查前移到方案生成与施工工况构造阶段，形成“拓扑—工况—内力—截面—配筋—门禁”的闭环。

## 2. 丰收湖项目暴露的根因

丰收湖异形基坑原计算出现 36 项 Fail、最大墙体位移 503.95 mm、最大墙体弯矩 5848.54 kN·m/m。根因由五个相互叠加的问题构成：

1. 短回墙和阶梯墙段没有直接支撑端点，被一维墙体模型按整段无支承竖向条带处理；
2. 标高 0.0 m 的第一道支撑在支撑层筛选中被错误忽略；
3. 拆撑和换撑阶段的竖向压力分带仅保留仍在场的临时支撑，已形成的楼板或永久围檩传力层未参与分带；
4. 闭合围檩的墙面端部被按自由悬臂端处理，角部连续传力未进入围檩包络；
5. 支点间距、有效无侧向支承长度和构件强度主要作为计算后检查，方案生成阶段缺少硬门禁。

## 3. P0：计算模型修复

### 3.1 异形墙段有效约束

短回墙与相邻主墙通过连续围檩和角部节点形成传力关系时，计算模型建立分析专用的角部传力约束。该约束不生成虚构施工构件，单独记录来源墙面、支撑层、传力路径和适用边界。

### 3.2 第一支撑零标高识别

支撑有效性按显式存在性和施工阶段激活状态判断。标高数值为 0.0 m 时继续作为有效支撑层处理，禁止使用布尔判断过滤数值零。

### 3.3 拆换撑压力分带

某施工阶段的竖向分担边界由以下集合共同建立：

- 当前在场临时支撑；
- 已拆临时支撑对应的永久楼板或换撑围檩层；
- 坑底和墙底边界。

永久传力层参与土压力分带，内力输出只保留该阶段真实存在的临时构件和永久构件，避免最上层支撑错误承担全部开挖侧压力。

### 3.4 闭合围檩多跨包络

闭合地下连续墙外围的围檩按角部刚接连续构件处理。设计包络包含：

- 直接支撑节点；
- 角部连续端约束；
- 各跨正弯矩；
- 支点负弯矩；
- 静力平衡反力；
- 闭式挠度估算。

系统同时检查围檩总反力与墙面线荷载合力的平衡误差。

## 4. P1：强度驱动设计闭环

设计流程调整为：

```text
初始围护与支撑拓扑
→ 墙段支撑覆盖检查
→ 围檩支点间距硬门禁
→ 角部扇形斜撑/局部短对撑增补
→ 非环形支撑交叉消除与 T/Y 节点临时立柱生成
→ 分阶段荷载路径同步
→ 墙、围檩、支撑、立柱承载力与刚度验算
→ 截面与配筋自动调整
→ 仍不满足时保留 Fail 并阻断施工图发行
```

### 4.1 围檩支点间距

项目设置包含建议上限和硬上限。超过硬上限时，系统先调整支撑拓扑，禁止仅依赖无限放大围檩截面掩盖过长受力跨。

### 4.2 有效无侧向支承长度

支撑长度检查采用墙端、临时立柱和 T/Y 节点形成的分段长度。总几何长度较长但已有可靠中间支承时，按最大服务跨验算。

### 4.3 非交叉 T/Y 网格节点

除明确的环梁—径向撑体系外，同层普通主对撑、次对撑和角撑不得在跨中相互穿越。次对撑或角撑遇到主对撑时，应在首个有效交点终止并形成 T/Y 节点；节点设置临时立柱，并在立柱服务记录中同时登记相关构件。两根连续杆件在交点两侧均延伸的情形一律作为硬失败。

### 4.4 自动恢复边界

系统允许自动执行：

- 短回墙局部支撑增补；
- 围檩超长支点间距修复；
- 局部短对撑和角部扇形斜撑；
- T/Y 节点临时立柱；
- 构件截面和初步配筋调整；
- 工况支撑 ID 和换撑层同步。

坡道、出土口、管线、邻近建筑、施工设备、节点详图和实际预加力缺失时，系统保留 Warning 或 Manual review，不自动假设现场条件。

## 5. P2：诊断、界面与证据链

### 5.1 诊断面板

计算页显示：

- 根因类别；
- 修复前后最大围檩支点间距；
- 新增支撑和共享节点数量；
- 强度、刚度和稳定性状态；
- 自动设计迭代上限；
- 仍需人工处理的项目边界。

### 5.2 结果证据

计算结果写入：

- `topologyPreflight`；
- `strengthDesignLoop`；
- `supportTopologySynchronization`；
- `replacementStageLoadPathPartition`；
- `closedPerimeterWaleEnvelope`；
- `sharedGridNodeRecovery`；
- 全局矩阵残差、条件数和正则化状态。

### 5.3 发行门禁

只要墙体、围檩、支撑、立柱、稳定性、几何拓扑或数值质量存在 Fail，施工图和正式报告发行状态保持禁止。自动修复成功只代表硬失败消除，项目仍需完成现场条件录入和专业审签。

## 6. 丰收湖项目修复结果

| 指标 | 修复前 | V3.14 修复后 |
|---|---:|---:|
| Fail | 36 | 0 |
| Warning | 16 | 5 |
| 最大墙体位移 | 503.95 mm | 3.418 mm |
| 最大墙体弯矩 | 5848.54 kN·m/m | 657.737 kN·m/m |
| 最大支撑轴力 | 8260.2 kN | 5261.129 kN |
| 强度状态 | Fail | Pass |
| 刚度状态 | Fail | Pass |
| 稳定性状态 | 未形成可发行结论 | Pass |

剩余 5 项 Warning 均属于实际工程输入或施工组织边界：第一道支撑工程含义、支撑端部与跨度复核、预加力和温度效应、换撑拆撑生命周期、坡道和出土口等障碍物。

## 7. 工程边界

V3.14 的计算结果可用于方案深化、专业复核和施工图前置设计。正式工程仍需通过经验证的有限元模型、勘察参数复核、节点专项计算、周边环境数据、施工组织、监测反馈和注册工程师审签。
