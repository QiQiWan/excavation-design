# 专业施工图生产链技术说明

## 1. 总体架构

```text
计算与深化模型
  → 图纸规则与视图计划
  → R2018 CAD 渲染器
  → 纸空间布局与原生标注
  → 钢筋加工与净距检查
  → CAD/PDF 独立验证
  → 施工版发行门禁
```

该架构将工程模型、图纸内容、出版样式和发行审批分离。图纸规则决定图种和视图，CAD 模板决定企业样式，质量门禁控制能否形成施工版。

## 2. CAD 数据约定

- 模型空间单位：mm。
- 模型空间比例：1:1。
- 默认 DXF：R2018/AC1032。
- 纸空间：A0—A4，视口锁定。
- 中文文字样式：`PIT_CN`。
- 标注样式：`PIT_DIM`。
- 审查版水印：`审查版 / AI-DRAFT / 不得直接用于施工`。

每张图在模型空间中保存工程实体，在纸空间中保存图框、图签、出版视口和审查状态。CAD 校验必须由独立解析器重新读取输出文件，不能只依赖生成器返回成功。

## 3. 图纸完整性规则

施工图最小集合包括：

- 图纸目录与总说明；
- 围护与支撑总平面；
- 控制剖面；
- 地连墙配筋图；
- 支撑配筋图；
- 围檩/冠梁配筋图；
- 节点大样；
- 钢筋表、BBS、加工分段、净距检查和签审清单。

项目没有对应构件时可豁免相应图种。规则包可以增加图纸，不能关闭工程最小集合。

## 4. 钢筋加工模型

钢筋对象依次经历：

1. 设计配筋组；
2. 逐根空间中心线；
3. 定尺/运输分段；
4. 接头类型和错开组；
5. 加工段和 BBS；
6. 净距与碰撞检查；
7. 图纸和材料表同步输出。

加工段长度满足：

\[
L_{piece}\leq \min(L_{stock},L_{transport})
\]

搭接钢筋的分段长度考虑搭接增量，机械连接考虑端部加工余量。接头位置采用错开组表达，项目深化阶段仍需结合墙幅分节、吊装方案和施工缝确定实际标高。

## 5. 正式发行门禁

施工版必须同时满足：

- 当前设计快照存在有效计算结果；
- 规范检查无硬失败；
- 配筋和加工无硬失败；
- DXF 全部通过版本、单位、纸空间、视口和 audit 检查；
- 最小图纸集合和加工表完整；
- 逐根几何无省略，预埋件碰撞状态有效；
- 设计、校核、审核、批准四级岗位分离完成；
- 当前施工版修订绑定当前设计快照；
- 输出 SHA-256 文件清单。

审查版可以在门禁未闭合时生成，用于发现问题和校审，不具备施工发行资格。

## 6. 外部系统边界

DXF 可以直接由系统生成和验证。DWG、CTB/STB 和企业打印机页面设置需要接入外部 CAD 运行环境。推荐发布流水线为：

```text
PitGuard R2018 DXF
 → ODA/AutoCAD 批量转 DWG
 → 企业 CTB/STB 批量打印
 → PDF 文本/字体/页面检查
 → 电子签章与归档
```
