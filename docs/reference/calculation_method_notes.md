# 计算方法说明（engineering-v0.2）

本版本把原 MVP 的 `TODO_ENGINEERING_FORMULA` 升级为可追溯的规范子集计算工作流。实现目标是形成可运行、可审查、可扩展的工程计算原型；它不是正式审图软件，所有结论仍需注册岩土/结构工程师复核。

## 已实现的计算子集

### 1. 水平荷载与土水压力

- 分层竖向总应力、有效应力积分。
- 朗肯主动土压力系数：`Ka = tan²(45° - φ/2)`。
- 朗肯被动土压力系数：`Kp = tan²(45° + φ/2)`。
- 静止土压力系数：`K0 = 1 - sinφ`，用于环境等级严格或位移控制要求较高的工况。
- 主动土压力：`pa = max(0, σ'v Ka - 2c sqrt(Ka)) + u`。
- 被动土压力：`pp = max(0, σ'v Kp + 2c sqrt(Kp)) + u`。
- 地下水压力按水土分算生成坑外/坑内水压力差。

### 2. 施工阶段与支撑轴力

- 自动生成支撑安装阶段和最终开挖阶段。
- 同一标高多道平行支撑按同一支撑层处理。
- 按土压力影响区积分得到墙后每延米线荷载，再按同标高平行支撑数量分配轴力。
- 设计效应记录 JGJ120/GB50009 子集中的重要性系数和组合系数。

### 3. 地连墙内力包络

- 采用分层压力曲线与支撑反力的等效竖向梁有限差分筛查模型。
- 输出剪力、弯矩、位移曲线及包络值。
- 设计值采用 `M_design = γ0 × 1.25 × M_standard`，`V_design = γ0 × 1.25 × V_standard`。

### 4. 稳定性筛查

- 嵌固稳定：按主动压力作用矩与嵌固段净被动抗力矩比值筛查。
- 坑底抗隆起：采用 Terzaghi/Meyerhof 型 `c-φ` 地基承载力因子筛查，`φ=0` 时退化为 `Nc=5.14, Nq=1` 的黏土极限。
- 抗渗流/突涌：按嵌固段有效自重与坑内外水头差筛查。
- 变形控制：按环境等级采用经验比例限值，正式限值应以项目审批文件、周边环境和 GB50007/JGJ120 适用条款复核。

### 5. 混凝土结构设计子集

- 地连墙正截面受弯：`M <= α1 fc b x (h0 - x/2)`，并按 `α1 fc b x = fy As` 求受拉钢筋。
- 地连墙受剪筛查：先按混凝土截面抗剪承载力子集复核，箍筋/水平筋构造另列。
- 支撑轴压筛查：`N <= φ(fc Ac + fy As)` 的矩形 RC 轴压子集。
- 最小配筋率、主筋直径、净距、水平筋间距、保护层等构造检查。

## 已接入的规则库目录

- `rules/jgj120_2012/`：水平荷载、嵌固稳定、抗隆起、抗渗流、变形、地连墙构造。
- `rules/gb50010/`：材料设计强度、正截面受弯、受剪、轴压、最小配筋率。
- `rules/gb50009/`：作用组合记录与设计效应转换。
- `rules/gb50007/`：基础/周边环境复核提示与占位筛查。
- `rules/gb55003_2021/`：地基基础通用规范安全提示、支护体系作用/稳定/变形/地下水控制复核要求。
- `rules/gb55008_2021/`：混凝土结构通用规范安全提示。
- `rules/gb50017/`：钢管支撑轴压强度/稳定筛查。
- `rules/enterprise/`：墙厚、墙深、支撑层数与标高初选规则。

## 仍需人工复核或后续增强

1. JGJ120 原文全部公式、全部支护形式、复杂水文地质、承压水、多层含水层和地方规定。
2. 圆弧滑动整体稳定、复杂坑底隆起模式、三维空间效应和邻近建筑/管线耦合。
3. 完整弹性地基梁参数反演、支撑刚度、预加力、换撑、拆撑和施工顺序细化。
4. GB/T50010 裂缝宽度、挠度、锚固、搭接、节点、抗震构造和施工图级详图。
5. 监测数据反分析和专家论证报告级审查流程。

计算书中会明确区分“已自动通过的子集筛查”“未通过项”“warning 项”和“manual_review 项”。
