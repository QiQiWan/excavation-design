# V1.1 优化迭代说明

## 已修复问题

1. **IFC 无法正常显示**
   - 原因：导出器将 `IFCEXTRUDEDAREASOLID` 的 `ExtrudedDirection` 写成 `$`。该字段在 IFC4 实体中应为挤出方向，很多 BIM Viewer 无法容忍空值。
   - 修复：统一输出 `IFCDIRECTION((0.,0.,1.))` 并引用到每个扫掠实体；同时修正钢筋局部坐标，避免轴线平行。

2. **水平支撑沿长边布设**
   - 修复：自动支撑改为跨越短向布置，即长边方向布置多榀短跨对撑，降低支撑长度、长细比和节点内力不确定性。

3. **同一面墙设计长度不统一**
   - 修复：连续共线边段被归为同一个 `DesignFace`，每片墙仍保留独立 segment 追溯，但共享 `DesignFaceCode`、`DesignLength` 和 `FaceSegmentIds`。

4. **围护结构超出地质模型范围**
   - 修复：创建/修改基坑、自动设计地连墙或支撑时，若基坑及 10m 缓冲范围超出现有 IDW 地质面，自动按钻孔重新外扩插值，并保留 warning，提示外推区域需补勘或人工确认。

5. **大长宽比基坑缺少角部约束**
   - 修复：长宽比达到阈值时，在四个直角附近生成 `corner_diagonal` 斜撑建议，并写入 IFC 属性集与前端 Viewer 属性。

## 真实基坑工程仍建议继续优化的细节

- 周边环境分区：邻近建筑、道路、管线、地铁、河道等应形成分区控制指标和差异化变形限值。
- 分区开挖和换撑：需支持分层、分块、栈桥、底板换撑、拆撑时序和工况自动拓扑更新。
- 支撑节点深化：围檩-支撑、角撑、立柱、格构柱、立柱桩、托座、预埋件、牛腿和节点局部承压应独立建模和校核。
- 降水与承压水：需要多含水层、水头时程、降水井、止水帷幕、坑内外水位差和渗流场耦合。
- 地质不确定性：应加入钻孔可信半径、透镜体/夹层、缺失地层解释、参数分项统计和不利剖面自动搜索。
- 施工偏差：围护墙垂直度、接缝渗漏、超挖、支撑预加力损失、温度效应和施工荷载移动需要进入组合。
- 监测反馈：位移、轴力、水位、沉降、支撑应力应支持导入并与计算包络对比，形成报警和参数反演。
- IFC 深化：后续可将钢筋由代表性组实体升级为按间距逐根生成钢筋笼，并输出节点、止水、预埋件和施工阶段状态。
